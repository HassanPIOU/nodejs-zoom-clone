# Node Js zoom clone


You can use NPM or Yarn. In my case I used Yarn

```bash
 git clone https://github.com/HassanPIOU/nodejs-zoom-clone
```

```bash
yarn install or npm install
```

```bash
 npm start
```

The default port is [3000](https://localhost:3000)
